# Mini-Carrer-Chatbot
A Python chatbot that suggests career paths based on your skills. Just tell the bot your skills, and it dynamically recommends relevant careers, along with helpful advice on learning and CV tips.

# Features
Suggests careers based on any skills mentioned, not just exact combinations.
Handles greetings, learning queries, and CV advice.
Easy to expand with new skills and career options via dictionaries.
Simple command-line interface for interactive conversations.

# Example Usage
You: I know Python and SQL
Bot: Based on your skills, you could consider: Data Analyst, Backend Developer, Machine Learning Engineer, Data Scientist, Database Administrator.
